# 项目 1：命令行 Todo 程序

## 目标

做一个能在终端里新增、查看、完成和删除任务的 Todo 程序，并把任务保存到本地文件。

## 最低要求

- 支持 `add <任务内容>`
- 支持 `list`
- 支持 `done <id>`
- 支持 `delete <id>`
- 程序重启后仍能读到之前保存的数据

## 建议拆分

1. 先定义 `Task` 结构体
2. 再设计命令行参数格式
3. 写加载任务的函数
4. 写保存任务的函数
5. 写 `add / list / done / delete` 对应逻辑
6. 最后整理输出文本和错误提示

## 进阶练习

- 增加 `help` 命令
- 支持按状态筛选任务
- 增加任务创建时间
- 把存储逻辑和命令逻辑拆到不同模块
- 给核心函数补测试

## 训练点

- `Vec<Task>`
- 模块组织
- `Result` 和 `?`
- 基础文件持久化
- 命令行参数处理

## 自测问题

- 哪些地方应该返回 `Result`
- 哪些地方不适合用 `panic!`
- 为什么 `list` 通常只借用数据
- 为什么保存文件逻辑最好单独拆出来
